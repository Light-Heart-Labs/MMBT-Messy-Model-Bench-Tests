# External Sources Used in This Deck

## Fonts
- **Helvetica Neue** — System font, no download required. Used for all headings and body text.
- **Arial** — Fallback for Helvetica Neue on Windows. System font.
- **Menlo** — Fallback for monospace numbers on Mac. System font.

## Icons/Imagery
- No external icons or imagery used. All visuals are data-driven charts.
- No clip-art, no stock photography, no AI-generated faces.

## Reference Data
- All data sourced from /input/repo/ (the investment memo repository).
- No external data was fetched during deck creation.

## Libraries
- **python-pptx** — Presentation generation
  - URL: https://github.com/scanny/python-pptx
  - Version: 0.6.23+
- **matplotlib** — Chart generation
  - URL: https://matplotlib.org/
  - Version: 3.9.2+

  - URL: https://pandas.pydata.org/
  - Version: 2.2+

  - URL: https://www.reportlab.com/
  - Version: 4.2+
