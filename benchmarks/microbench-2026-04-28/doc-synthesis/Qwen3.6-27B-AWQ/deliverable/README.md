# Nimbus Logistics — Follow-On Investment Brief

## How to Read This Output

### Primary Deliverable

- **`brief.md`** — The 1-page executive brief. Start here. It contains a clear PASS recommendation with supporting analysis, source citations, and conditions under which the recommendation would change.

### Supporting Materials

- **`key-facts.md`** — A complete list of every material fact incorporated into the brief, with source attribution. Use this to verify that no numbers were invented and all claims trace to the five sources.

- **`research/notes.md`** — Working notes from reading the five sources. Contains source-by-source summaries, key tensions identified, and source quality assessments.

- **`research/dead-ends.md`** — Ideas that were considered during analysis but not included in the brief, with explanations for why they were abandoned.

- **`decisions/adr-001-recommendation-stance.md`** — Architecture Decision Records explaining non-obvious framing choices:
  - Why the recommendation is PASS rather than "more diligence needed"
  - How the leaked memo was weighted relative to other sources
  - Why the brief is structured around red flags rather than balanced pros/cons

### Methodology

1. Read all five sources in full.
2. Extract every material fact with source attribution.
3. Identify tensions and contradictions between sources.
4. Assess source quality and reliability.
5. Synthesize into a clear recommendation with supporting analysis.
6. Verify every number traces to a source — no invented metrics.
7. Keep the brief under 700 words.
