# Narrative Alternatives Considered And Rejected

## Alternative 1: Memo Redelivery

Structure: restate recommendation, thesis, valuation, risks, and conclusion in the same order as the memo.

Rejected because the board has read the memo. This would show the stock pick but not the agent's reasoning system.

## Alternative 2: Audit-First Forensic Walkthrough

Structure: begin with source hashes, extraction steps, workbook checks, and traces; reveal the recommendation late.

Rejected because the prompt requires leading with the recommendation. It also risks turning a 30-minute discussion into a compliance review before the board understands the investment call.

## Alternative 3: Bull/Bear Debate

Structure: frame the deck as a debate between "international upside" and "tariff/category risk."

Rejected because it underplays the most interesting capability demonstration: the agent declined to force a sell-side miss and documented why.

## Selected Structure

Lead with the recommendation, then move from investment conclusion to reasoning integrity:

1. What the agent recommended.
2. Why the recommendation is HOLD, not BUY or SELL.
3. What evidence drove the call.
4. How uncertainty changes the target distribution.
5. How the agent reasoned from files to decisions to conclusion.
6. What the agent rejected.
7. How a board member can audit any claim.
8. What remains uncertain.
