# ADR 0001: Lead With Recommendation, Then Prove The Reasoning Trail

Date: 2026-04-27

## Status

Accepted.

## Context

The prompt requires leading with the recommendation, while also emphasizing that the board cares about the agent system's reasoning more than the stock pick.

## Alternatives Considered

- Memo redelivery: clear but redundant and weak as a capability demonstration.
- Audit-first walkthrough: strong on traceability but violates the prompt's "lead with the recommendation" requirement.
- Bull/bear debate: lively but hides the process evidence.

## Decision

Use a 20-slide arc that opens with the `HOLD / $41` recommendation, then progressively shows evidence, model mechanics, scenario uncertainty, reasoning graph, dead ends, self-audit workflow, and limitations.

## Consequences

The deck is not a dense diligence appendix. It is a discussion instrument that lets the board interrogate both the conclusion and the agent's reasoning discipline.
